    • Kyoko settings
      1. token: your Bot Token
      2. yt_api_key: your Youtube api key
      3. admin_id: your client id
      4. bot_category: config the channel category
      5. botchannel: config the bot channel
      6. defaultVolume: the player default Volume
      7. MaxQueue: Max Songs in Queue
      8. BotName: You can give the bot a new Name
      9. Delete_Message: thats a auto purge message delete option
      10. songList_25 - songList_50: songlist size, 25 or 50 stores per account
      11. prefix: start bot command
      12. debugBot: debug option
      13. help_send_privat: send the help message private or in bot channel
      14. songlist_send_privat: send the songlist message private or in bot channel
      15. language option: config you language
      16. send_emoji_bar: send the emoji bar true or false
        17. -------------------------------------------------
        The function to save and delete songs is completely private. This is extra for the bot channel to stay clear.

    • How do I configure the emoji bar?

      You can use this emoji bar to operate the player.
      If you do not need this, you can switch this function on/off in the setting.

      the emojis will be uploaded, [discord, server-settings, emoji upload].
      the emoji id can be send with the admin command [prefix emojis].
      You must then enter this id under bot settings - emoji setting.

    • Admin Funktions

        you can use the command [prefix admin] to display all admin commands.
        this message will be sent privately.

      1. admin: listening the admin commands
      2. emojis: get the custom emoji id
      3. user_info: get all user info on server
      4. restart: restart the bot
      5. install: auto install the channel category and the bot channel
