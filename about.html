  • Kyoko settings

    1. token: your Bot Token
    2. yt_api_key: your Youtube api key
    3. admin_id: your client id
    4. botchannel: in which channel is the bot divided
    5. defaultVolume: the player default Volume
    6. MaxQueue: Max Songs in Queue
    7. BotName: You can give the bot a new Name
    8. Delete_Message: thats a auto purge message delete option
    9. songList_25 - songList_50: songlist size, 25 or 50 stores per account
    10. prefix: start bot command
    11. debugBot: debug option
    12. help_send_privat: send the help message private or in bot channel
    13. songlist_send_privat: send the songlist message private or in bot channel
    14. language option: config you language
    15. send_emoji_bar: send the emoji bar true or false
      16. -------------------------------------------------
      The function to save and delete songs is completely private. This is extra for the bot channel to stay clear.
      -------------------------------------------------
      In the commands-setting you can customize the bot commands.

  • Admin Funktions

      you can use the command [prefix admin] to display all admin commands.
      this message will be sent privately.

    1. admin: listening the admin commands
    2. emojis: get the custom emoji id
    3. user_info: get all user info on server
    4. restart: restart the bot
    5. install: auto install the channel category and the bot channel
